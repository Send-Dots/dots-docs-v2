---
title: 'Retrieve an Invoice'
---

<Note>
  All requests to the `/v2/invoice/*` endpoints must include both the
  `client_id` and `api_key` as detailed in the
  [Authentication](/overview/authentication) document.
</Note>

Get an invoice by its ID

### Method

Make a request to `/v2/invoice/get/{INVOICE_ID}`

### Request Example

<CodeGroup>

```bash cURL
curl https://pls.senddotssandbox.com/invoice/get/8f246ca0-4ade-4296-bab4-5a6d37d9af10 \
-H "Authorization: Basic <TOKEN>"
```
