A `transaction` represents a movement of funds on the Dots platform. To create a `transaction` you must create a transfer.

## The Transaction Object

### Example:

```json
{
  "id": 12341,
  "amount": 10000,
  "created": "2019-01-01T00:00:00.000Z",
  "source_name": "App",
  "destination_name": "User",
  "metadata": {}
}
```
